# Type 7 Arcade MVP

Browser MVP build for the Type 7 arcade demo.

## Local Run
```bash
cd "/Users/nattwiss/Desktop/Type 7 Outrun Game/versions/MVP_Final"
PATH="/Users/nattwiss/Desktop/Holocene Calendar/tools/node20/bin:$PATH" npm run dev
```

## Build
```bash
cd "/Users/nattwiss/Desktop/Type 7 Outrun Game/versions/MVP_Final"
PATH="/Users/nattwiss/Desktop/Holocene Calendar/tools/node20/bin:$PATH" npm run build
```

## Controls
- `A/D` or `←/→`: steer
- `W/S` or `↑/↓`: throttle / brake
- `R`: restart run
- `Q`: toggle hidden debug/tools
